#Github1
##1. GitHub  是什么 
GitHub 是一个IT技术人员的社交网站。
是一个面向开源及私有软件项目的托管平台，只支持 Git 作为唯一的版本库格式进行托管。 除了 Git 代码仓库托管及基本的 Web 管理界面以外，还提供了订阅、讨论组、文本渲染、在线文件编辑器、协作图谱（报表）、代码片段分享（Gist）等功能。
##2. G it 
Git是一款免费、开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。
